# Flappy Bird Game

A fun and interactive implementation of the classic Flappy Bird game built with HTML5, CSS3, and JavaScript.

## Overview

This is a browser-based recreation of the popular Flappy Bird mobile game. Navigate your bird through pipes by clicking or tapping to make it jump, and try to get the highest score possible!

## Features

- 🎮 **Interactive Gameplay** - Click to make the bird jump and avoid pipes
- 🎯 **Score Tracking** - Keep track of your score as you progress
- 📱 **Responsive Design** - Works on different screen sizes
- 🎨 **Clean Graphics** - Beautiful gradient background and smooth animations
- 🔄 **Game Over & Restart** - Play multiple rounds

## How to Play

1. Open `index.html` in your web browser
2. Click the "Start Game" button to begin
3. Click anywhere on the screen (or press spacebar) to make the bird jump
4. Navigate through the pipes without hitting them
5. Each pipe you successfully pass increases your score by 1
6. If you hit a pipe or the ground, the game ends
7. Click "Restart" to play again

## Technologies Used

- **HTML5** - Game structure and canvas for rendering
- **CSS3** - Styling and animations
- **JavaScript (Vanilla)** - Game logic and mechanics
